# Sean's Blog - AI 技术分享

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/git/external?repository-url=https://github.com/wersling/wersling-blog)

## 关于本博客

这是 Sean 的个人技术博客，专注于分享 AI、RAG、机器学习、深度学习、编程技术和个人成长等内容。我致力于探索人工智能前沿技术，记录开发实践经验。

**博客地址**: [https://blog.wersling.cn](https://blog.wersling.cn)

## 技术栈

本博客基于以下技术构建：

- **框架**: [Next.js 14](https://nextjs.org/) (App Router)
- **样式**: [Tailwind CSS 3.0](https://tailwindcss.com/)
- **语言**: [TypeScript](https://www.typescriptlang.org/)
- **内容管理**: [Contentlayer](https://www.contentlayer.dev/) - 将 MDX 文件转换为类型安全的 JSON 数据
- **部署**: [Vercel](https://vercel.com/)
- **评论系统**: [Giscus](https://giscus.app/) (基于 GitHub Discussions)
- **分析工具**: Google Analytics
- **订阅服务**: Buttondown

## 主要特性

- ✅ 响应式设计，支持移动端和桌面端
- ✅ 明暗主题切换
- ✅ MDX 支持，可在 Markdown 中编写 JSX
- ✅ 代码高亮，支持行号和行高亮
- ✅ 标签系统，每个标签都有独立页面
- ✅ 文章搜索功能 (Kbar)
- ✅ RSS 订阅
- ✅ SEO 优化，自动生成 sitemap
- ✅ 评论功能 (Giscus)
- ✅ Newsletter 订阅
- ✅ 数学公式支持 (KaTeX)
- ✅ 图片自动优化
