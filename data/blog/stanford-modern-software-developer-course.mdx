## 不写一行代码！

这是斯坦福大学计算机系新课[CS146S: The Modern Software Developer](https://themodernsoftware.dev/)对全体学生的硬性要求。这门课在今年9月秋季学期首次上线就被挤爆。

课程开发人 Mihail Eric 在第一堂课就立下规矩：整个学期的开发任务必须主要依赖 AI 完成，学生在 Git 提交代码时，必须附带与 AI 的对话记录以供核查。

开课仅两个多月，就有很多学生成功构建并发布了完整的全栈应用产品。

现在，这门课材料现已在网上全面发布！

课程主页：https://themodernsoftware.dev/

每周配套的 Reading List（阅读清单）推荐阅读，精选了很多业界一线团队写的技术博客和论文，既有实践细节，也能看出一些方法论。

对想系统了解「AI 软件开发」的学习者来说，即便不完整上课，只顺着这些材料读一遍，也能建立起比较成形的框架。

斯坦福这门课程的推出，是一个信号——软件开发的范式转换已经开始。1-2年后，我们今天熟悉的编程方式可能会成为"古典编程"。

你准备好了吗？
